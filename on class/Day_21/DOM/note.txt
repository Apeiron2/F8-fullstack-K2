#DOM

## DOM Element
- Truy xuất vào các thẻ HTML => Tạo ra Node(Nút)
- node là object

## DOM HTML

- Thay đổi nội dung và thuộc tính HTML
- Thêm, sửa, xóa, đọc

## DOM CSS

- Thay đổi CSS inline

## DOM Navigation

- Thực hiện truy xuất vào các thẻ html theo phân cấp cha con

## DOM Event

- Gán sự kiện vào thẻ HTML

## DOM Event Listener

- Giúp lắng nghe các sự kiện

## DOM Node

- Thao tác với các Element thông qua object

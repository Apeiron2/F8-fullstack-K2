# Redux

    - Store
        Kho chứa state global
    - Reducer
        Hàm cập nhật State
    - Action
        Object giúp mô tả các hành động từ UI -> Reducer
    - Dispatch
        Hàm gửi action từ UI -> Reducer
    - Subscribe
        Lắng nghe sự thay đổi state trên store
